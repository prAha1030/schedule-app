## Body

### 일정 (Schedule)

**일정 생성**
- 일정 제목, 일정 내용, 작성자명, 비밀번호 요청
- 고유 식별 번호, 일정 제목, 일정 내용, 작성자명 저장 및 응답
- 작성일, 수정일, 비밀번호 저장

**일정 조회**
- 일정 단건 조회 
  - 일정의 고유 식별 번호를 통해 조회
- 일정 다건 조회 
  - 일정 전부 조회 (작성자명을 통해 일정 다건 조회 가능)
  - 수정일 기준으로 내림차순

**일정 수정**
- 일정 제목, 작성자명만 수정 가능
- 수정 시 비밀번호 필요

**일정 삭제**
- 삭제 시 비밀번호 필요

### 댓글 (Comment)

**댓글 생성**
- 댓글 내용, 작성자명, 비밀번호 요청
- 일정 고유 식별 번호, 댓글 내용, 작성자명 저장 및 응답
- 작성일, 수정일, 비밀번호 저장
- 하나의 일정에 최대 10개의 댓글까지 가능

**일정 단건 조회 with 댓글**
- 일정 단건 조회 시, 해당 일정의 댓글들도 함께 응답

## Tail

- 최초 생성 시 수정일은 작성일과 동일하다.
- 비밀번호는 모든 응답에서 제외되어야 한다.
- 일정 수정, 삭제 시 비밀번호 일치해야 수정, 삭제 가능
- 일정 제목은 최대 30자, 일정 내용은 최대 20자, 댓글 내용은 최대 100자 이내로 제한!
- 일정 제목, 일정 내용, 댓글 내용, 비밀번호, 작성자명은 필수 입력값!
